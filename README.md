# Weka Java

Estudo de caso de Machine Learning utilizando Java, JavaCV (OpenCV) e Weka API para classificação binária de personagens de desenho animado.

## Algorítimos utilizados

* NaiveBayes
* J48
* OneR
* JRip
* IBK
* LibSVM
* MultiLayerPerceptron
